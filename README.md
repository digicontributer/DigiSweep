## Install

## Scripts

 - **npm run start** : `react-scripts start`
 - **npm run build** : `react-scripts build`
 - **npm run test** : `react-scripts test`
 - **npm run eject** : `react-scripts eject`
 - **npm run predeploy** : `npm run build`
 - **npm run deploy** : `gh-pages -d build`
